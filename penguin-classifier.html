Penguin Species Classifier

Classification model that identifies penguin species based on physical measurements using machine learning algorithms for accurate species prediction.

View on GitHub

Overview

This project builds a multi-class classification model to identify penguin species based on physical measurements. Using the Palmer Penguins dataset, the model analyzes bill length, bill depth, flipper length, and body mass to distinguish between three penguin species: Adelie, Chinstrap, and Gentoo.

The project explores multiple classification algorithms including Logistic Regression, Random Forest, and Support Vector Machines (SVM) to find the best performing model. It includes comprehensive exploratory data analysis, feature engineering, and model comparison to achieve high classification accuracy.

Key Features

  • Multi-Class Classification: Distinguishes between 3 penguin species
  • Multiple Algorithms: Compares Logistic Regression, Random Forest, and SVM
  • Exploratory Data Analysis: Visualizes relationships between features and species
  • Feature Engineering: Creates derived features for improved accuracy
  • Model Comparison: Systematic evaluation of algorithm performance
  • Interactive Predictions: Input measurements to predict species

Dataset

The Palmer Penguins dataset contains 344 observations of three penguin species collected from islands in the Palmer Archipelago, Antarctica. The dataset is an excellent alternative to the classic Iris dataset for teaching classification.

Species:

  • Adelie: 152 samples - Small penguins with short bills
  • Chinstrap: 68 samples - Medium penguins with distinctive chin strap
  • Gentoo: 124 samples - Largest penguins with long bills

Features Used:

  • Bill Length (mm): Length of the penguin's bill
  • Bill Depth (mm): Depth/thickness of the bill
  • Flipper Length (mm): Length of the flipper
  • Body Mass (g): Weight of the penguin

Model Performance

Algorithm Accuracy Precision Recall
Random Forest 98% 98% 98%
SVM 97% 97% 97%
Logistic Regression 95% 95% 95%

Random Forest achieved the best performance with 98% accuracy, likely due to its ability to capture non-linear relationships between features.

Exploratory Data Analysis

Comprehensive visualization revealed clear separability between species based on physical measurements:

  • Bill Length vs Depth: Gentoo penguins have distinctly longer, shallower bills
  • Flipper Length: Gentoo penguins have significantly longer flippers
  • Body Mass: Gentoo penguins are substantially heavier
  • Adelie vs Chinstrap: Similar body size but different bill characteristics

Pair plots and correlation analysis showed that flipper length and body mass are the strongest predictors for distinguishing Gentoo from other species.

Implementation Highlights

Data Preprocessing

  • Removed missing values (11 observations with incomplete measurements)
  • Encoded categorical variables (species, island, sex)
  • Standardized features for algorithms sensitive to scale (SVM, Logistic Regression)
  • Stratified train-test split to maintain class proportions

Model Training

  • 5-fold cross-validation for robust performance estimation
  • Hyperparameter tuning using GridSearchCV
  • Feature importance analysis for Random Forest
  • Decision boundary visualization for 2D feature combinations

Model Selection

  • Compared algorithms on accuracy, precision, recall, and F1-score
  • Analyzed confusion matrices to identify misclassification patterns
  • Selected Random Forest as final model due to best overall performance
