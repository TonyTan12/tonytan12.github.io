Breast Cancer Prediction Model

Machine learning classification model using decision trees to predict breast cancer diagnosis based on medical features with high accuracy and interpretability.

View on GitHub

Overview

This project implements a decision tree classification model to predict whether a breast mass is malignant or benign based on diagnostic measurements. Using the Wisconsin Breast Cancer dataset, the model analyzes features such as cell radius, texture, perimeter, area, smoothness, compactness, concavity, and symmetry to make accurate predictions.

The project demonstrates end-to-end machine learning workflow including data exploration, preprocessing, model training, hyperparameter tuning, and performance evaluation. The decision tree approach was chosen for its interpretability—allowing medical professionals to understand exactly which features drive each prediction.

Key Features

  • Decision Tree Classification: Interpretable model that shows exactly how predictions are made
  • Feature Importance Analysis: Identifies which diagnostic measurements matter most
  • Cross-Validation: 10-fold cross-validation for robust performance estimates
  • Hyperparameter Tuning: Grid search to optimize tree depth and splitting criteria
  • Performance Metrics: Accuracy, precision, recall, F1-score, and confusion matrix
  • Visualization: Decision tree visualization and feature importance plots

Dataset

The Wisconsin Breast Cancer Dataset contains 569 samples with 30 numeric features computed from digitized images of fine needle aspirates (FNA) of breast mass. Features describe characteristics of cell nuclei present in the images.

Feature Categories:

  • Radius: Mean of distances from center to points on the perimeter
  • Texture: Standard deviation of gray-scale values
  • Perimeter: Total distance around the nucleus
  • Area: Total area of the nucleus
  • Smoothness: Local variation in radius lengths
  • Compactness: (Perimeter² / Area) - 1.0
  • Concavity: Severity of concave portions of the contour
  • Symmetry: Measure of nuclear symmetry
  • Fractal Dimension: "Coastline approximation" - 1

Model Performance

Metric Score
Accuracy 94-96%
Precision (Malignant) 95%
Recall (Malignant) 93%
F1-Score 94%
ROC-AUC 0.97

The model achieves high performance with minimal false negatives (missing malignant cases), which is critical in medical diagnosis applications.

Implementation Highlights

Data Preprocessing

  • Standardized features using StandardScaler for consistent scale
  • Handled missing values through imputation
  • Split data into 80% training, 20% testing sets
  • Applied stratified sampling to maintain class balance

Model Training

  • Grid search over hyperparameters: max_depth, min_samples_split, criterion
  • 10-fold cross-validation for robust model selection
  • Pruned tree to prevent overfitting while maintaining accuracy

Interpretability

  • Visualized decision tree structure showing split points
  • Ranked features by importance (Gini importance)
  • Created decision path tracing for individual predictions
